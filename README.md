# Gender Classification (Python)

This project builds a machine learning model to classify gender based on structured dataset features. The workflow includes data preprocessing, model training, evaluation, and performance analysis.

## Tech Stack
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  

## Key Work
- Data cleaning and preprocessing  
- Train-test split and feature scaling  
- Logistic Regression model implementation  
- Performance evaluation using accuracy, precision, recall, F1-score  
- Confusion matrix visualization  

The notebook demonstrates an end-to-end classification pipeline from dataset loading to model evaluation, achieving ~97% accuracy.
